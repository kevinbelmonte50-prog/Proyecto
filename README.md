# Gestor de Tareas en Python

Este proyecto es un gestor de tareas desarrollado en Python como trabajo final para la materia Programación 1.

El programa permite administrar tareas desde la consola utilizando un sistema simple de gestión.

## Funcionalidades

- Agregar tareas
- Listar tareas
- Marcar tareas como completadas
- Eliminar tareas
- Guardar tareas en archivo JSON

## Tecnologías utilizadas

- Python 3
- Librería tabulate
- Entorno virtual (venv)

## Instalación

1. Clonar el repositorio

2. Crear entorno virtual

python -m venv venv

3. Activar entorno virtual

Windows

venv\Scripts\activate

4. Instalar dependencias

pip install -r requirements.txt

## Ejecutar el programa

python main.py
